Getting Started
===============

#. Read the Crazyswarm paper to understand the different components of the system. A preprint is available `here <http://usc-actlab.github.io/publications/Preiss_ICRA2017.pdf>`_.
#. Learn more about the Crazyflie platform, e.g. by reading sections 1 and 2 of "Flying Multiple UAVs Using ROS", Springer Book on Robot Operating System (ROS), 2017. A preprint is available `here <http://usc-actlab.github.io/publications/Hoenig_Springer_ROS2017.pdf>`_.
#. Learn about the Robot Operating System (ROS), e.g. by reading part 1 of "Programming Robots with ROS. A Practical Introduction to the Robot Operating System" by Morgan Quigley, Brian Gerkey, William D. Smart, O'Reilly, 2015.
#. Get to know your motion capture system (such as VICON or OptiTrack) by reading the respective user manuals.
